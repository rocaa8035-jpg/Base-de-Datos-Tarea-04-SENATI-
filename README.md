# Base-de-Datos-Tarea-04-SENATI-
En este repositorio quiero que conozcan mi trabajo en esta tarea sobre "Base de datos".

## 📋 Requisitos Previos

Para ejecutar este proyecto, necesitas tener instalado:

- [XAMPP](https://apachefriends.org) (incluye Apache y MySQL)
- Navegador web moderno.

## 🚀 Instalación y Configuración

1.  **Clonar el repositorio:**
    ```bash
    git clone https://github.com
    ```
2.  **Mover los archivos:**
    Mueve la carpeta del proyecto a `C:\xampp\htdocs\`.
3.  **Configurar Base de Datos (MySQL):**
    - Inicia XAMPP y activa **Apache** y **MySQL**.
    - Abre tu navegador y ve a `http://localhost/phpmyadmin`.
    - Crea una nueva base de datos llamada `bdvet`.
    - Importa el archivo `Tarea04_script.sql`.
4.  **Configurar Conexión:**
    - Edita el archivo de configuración de conexión a BD (ej. `config.php`) con las credenciales (usuario: `root`; sin contraseña).
5.  **Ejecutar:**
    - Abre en tu navegador: `http://localhost/tu-repo/`.
